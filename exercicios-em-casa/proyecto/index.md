# Repositorio de Estudio: JavaScript (Frontend) + Node.js (Backend)

Este repositorio esta disenado para ayudarte a dominar los fundamentos de JavaScript y Node.js de forma practica, progresiva y orientada al mundo real.

## Como usar este repositorio

1. Lee cada archivo en orden dentro de cada modulo.
2. Ejecuta y modifica los ejemplos de codigo para experimentar.
3. Resuelve los ejercicios practicos al final de cada concepto.
4. Intenta completar el desafio de cada tema sin mirar soluciones externas.
5. Construye el proyecto final del modulo para consolidar el aprendizaje.

## Metodologia de cada archivo de concepto

Cada archivo de estudio seguira siempre esta estructura:

1. **Titulo y Definicion**: que es y para que sirve.
2. **Sintaxis y Ejemplos de Codigo**: ejemplos claros y comentados.
3. **Evolucion y Contexto Profesional (Senior Tip)**: antes vs ahora, practicas en desuso y estandar actual.
4. **Ejercicios Practicos**: 2 o 3 ejercicios cortos.
5. **Desafio del Concepto**: un reto mas cercano a un escenario real.

## Orden recomendado

- Comienza por `modulo_1_javascript_frontend/`.
- Luego avanza a `modulo_2_nodejs_backend/`.
- Finaliza cada modulo con su `PROYECTO_FINAL.md`.

## Objetivo final

Al terminar ambos modulos deberias poder:

- Escribir JavaScript moderno en frontend con buenas practicas.
- Construir una API REST en Node.js y Express.
- Conectar backend a base de datos.
- Validar datos de entrada y organizar codigo modularmente.
